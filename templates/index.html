PROJECT PORTFOLIO

Ax=xB

CALIBRATION API

The pmCalibrate API is an open source tool for calibrating wrist mounted cameras on robotic arms. It uses the Park and Martin method to find solutions to the Ax=xB kinematic system.

HOPHACKS 2017:

HEALTH.IO

Health.io is an app that utilizes machine learning to predict which hospitals are most suited to treat a specific patient. Health.io runs on a neural net I wrote in tensorflow and trained on the John Snow Hospital Comparison Database. Health.io won the Google Prize at Hophacks 2017.

BITCAMP 2016:

SCRIBE

Scribe is an optical character recognition (OCR) robot. Scribe runs on a custom linux kernel embedded on the Intel Edison microprocessor, and utilizes a random forest classifier for its OCR processing. Scribe won the Intel Grand Prize at Bitcamp 2016.

THIS

WEBSITE

I wrote this website entirely from scratch in order to learn web development. It utilizes python-flask for the back end, gunicorn for scalable deployment, heroku for hosting, and HTML5, CSS3, JQuery, and Bootstrap for the front end.

MLH PRIME 2016:

CROWDIFY

Crowdify is a distributed jukebox application that lets users at a party or in a coffee shop vote on which songs should be played. I wrote the entire back-end for crowdify in under 24 hours, and the app won a top 10 prize at MLH Prime 2016.
